# 电商系统用户操作说明（dulizhanshop）

## 概况

- 主要功能：
  - 管理后台：控制台总览、产品管理、分类管理、导航管理、首页内容与轮播图、站点设置、SEO 设置、统计代码、消息与邮箱管理、数据重置、修改密码。
  - 前台站点：商品列表与详情页、搜索与筛选、评分与评论展示、首页轮播与文案、导航与分类浏览。
  - 邮箱收集与导出：可收集用户邮箱与留言，并支持批量下载。
- 优势：
  - 商品详情采用亚马逊风格布局与信息密度（多图、规格参数、评价区块、交叉销售推荐），转化导向强、可读性好。
  - 支持收集用户邮箱与留言，并可按需批量导出，适合做 EDM 营销与客户回访。
  - 支持 SEO：标题、关键词、描述可配置；站点验证（Google/Baidu）；Sitemap/Robots 规则可配置。
  - 支持第三方统计脚本：可在后台 Analytics 页面粘贴统计代码，适配常见平台。
  - 首页加载快、体验稳定：适合电商推广与投放。
  - 模块化、易扩展：管理后台各模块相互独立，便于后续功能扩展与权限细分。

## 使用说明
1、入口与登录
 - 管理后台地址：你的域名`/admin`
 - 默认管理员：用户名 `dage666`，密码 `dage168`
 - 登录成功进入后台首页（仪表盘），右上角可“修改密码”和“退出登录”
 - 建议首次登录后立即修改密码

2、常用菜单
 - `Products` 产品管理
 - `Categories` 分类管理
 - `Navigation` 导航菜单
 - `Home Content` 首页文案
 - `Carousel` 轮播图管理
 - `SEO` SEO 设置
 - `Analytics` 统计代码
 - `Settings` 站点设置
 - `Messages` 留言/联系表单
 - `Reset` 数据重置

3、产品管理（按照亚马逊标准）
 - 新建产品：Admin → Products → New
 - 关键字段：
   - 标题（Title）：用于生成产品地址（slug）
   - 价格（Price）与原价（Original Price）
   - 分类（Category）：需先创建分类再选择
   - 主图与图片集（Images）：至少一张
   - 亚马逊链接（Amazon URL）：支持不同变体对应链接
   - 要点（Bullet Points）：建议填写 5 条
   - 描述（Description）：长描述
   - 底部按钮显示：Show Buy on Amazon、Show Add to Cart
   - 精选（Featured）：勾选后可显示在首页精选
  - 变体（Variants）：
    - 添加变体组：例如颜色（Color）、尺寸（Size）等，每组包含若干选项值
    - 选项主图：为各选项配置主图缩略图，前台选择该选项时可优先展示对应图片（映射到图片集）
    - 图片关联：将选项与图片集关联，选择选项会自动切换至对应主图
    - 购买链接：可为每个选项填写购买链接；当所有分组均已选择时，优先跳转“组合链接”；否则按最近点击的分组或任一已选分组的链接回退，最后兜底到产品主链接
    - 前台效果：用户在详情页选择不同选项，主图与“Buy on Amazon”链接会随之更新
    - 保存：完成配置后点击保存，前台即时生效
 - 编辑/删除：列表页进入某产品 → Edit 或执行删除
 - 上架控制：`Active` 决定是否在前台可见
 - 首页精选说明：
   - 首页只展示最新的 3 个精选产品
   - 勾选精选超过 3 个也没关系，首页只显示 3 条，其余仍在“全部产品”列表可见

4、分类管理（Categories）
 - 新建分类：Admin → Categories → New
 - 字段：名称（Name）、别名（Slug）、描述和封面图（可选）
 - 编辑/删除：在列表点击进入详情操作
 - 建议先建分类，再建产品并选择分类

5、导航管理（Navigation）
 - 菜单项：名称（label）、链接（href）、排序（order）、启用状态（active）
 - 前台显示规则：
   - 中间菜单不显示购物车项
   - 右侧固定显示购物车入口
   - 可添加外部链接，`href` 支持完整 URL

6、首页内容（Home Content）
 - 编辑首页大标题、副标题、三张特色卡片（标题与描述）
 - 修改后前台首页即时生效

7、站点设置（Settings）
 - 站点名称（Site Name）
 - Logo 地址（Logo URL）
 - SEO 描述与关键词（Site Description、Keywords）
 - 联系方式（Email、Phone、Address）
 - 社媒链接（Facebook/Twitter/Instagram/YouTube）
 - 页脚文案（Footer Text）
 - 关于我们/使命/为什么选我们等文案
 - 保存即生效，建议与首页内容配合调整

8、消息管理（Messages）
 - 查看用户通过“联系我们”表单提交的留言
 - 支持标记“已读”，敏感信息请及时处理并清理
 - 顶部提供“邮箱管理”入口，点击进入邮箱列表页面（支持搜索、复制与下载）

9、账号与安全
 - 修改密码：Admin → Change Password
 - 退出登录：右上角 Logout
 - 会话有效期：约 7 天；长时间不操作需重新登录
 - 请勿在任何产品或页面字段中填写密钥、数据库地址等敏感信息

10、数据重置（Reset）
 - 入口：Admin → Reset
 - 模式：
   - Data：重置数据（导航、分类、产品、首页内容、站点设置、消息），保留当前管理员与会话
   - Full：重置所有数据并重置管理员为默认账号，同时登出
   - Password：仅重置当前管理员密码为默认值（不影响数据）
 - 重置后会自动注入预设数据，首页精选为 3 个，保持与初始一致

11、SEO 设置（SEO）
 - 入口：Admin → SEO
 - 内容：
   - Sitemap：可开启/关闭；设置更新频率与优先级；选择是否包含产品与分类
   - Robots：推荐屏蔽后台（/admin）、接口（/api）、购物车（/cart）、结算（/checkout）、搜索页（/products?q=...）等
   - 站点验证：可填写 Google / Baidu 验证值，用于搜索引擎站点所有权验证
 - 提示：
   - 站点标题、关键词与描述在“Settings”中配置（Site Name、Description、Keywords）
   - SEO 与验证设置保存后前台将即时生效，无需重新部署

12、轮播图（Carousel）
 - 入口：Admin → Carousel
 - 新建轮播项：上传图片（建议 ≤5MB），设置标题、按钮文字、跳转链接，可选择“新窗口打开”
 - 排序与启用：支持拖拽或上下移动调整顺序；可设置是否启用（active）
 - 编辑/删除：可更新图片与文案，删除后从首页轮播移除
 - 建议：为移动端与桌面端选择合适尺寸图片，尽量压缩体积以提升加载速度

13、搜索（Search）
 - 前台搜索框：在网站顶部导航的“搜索”输入关键词，点击搜索，即可在“产品列表”页展示匹配结果
 - 分类筛选：在产品列表页顶部下拉选择分类后应用筛选；支持同时携带搜索关键词
 - 移动端：展开导航菜单后，支持在弹出层内进行搜索与按分类浏览
 - 结果展示：支持评分与评论数量的聚合显示，便于用户对比选择

14、评论与评分（Reviews）
 - 前台展示：产品卡片与详情页将展示平均评分与评论数量；详情页支持“按评分/按时间”排序，支持“只看带图”筛选；图片评论可点击放大预览
 - 后台管理：在产品编辑页的“产品评论”区域可添加/编辑评论，设置星级与配图，控制“是否显示在前台”
 - 字段说明：昵称（Name）、国家（Country）、标题（Title）、正文（Content）、星级（Rating）、是否显示（显示在前台）、图片（可选）、发言时间（可自定义）
 - 操作步骤：
   - 添加评论：点击“添加评论”，填写上述字段，设置“显示在前台”，选择“发言时间”（可自定义日期与时间），保存
   - 编辑评论：在已有评论列表中选择“编辑”，可修改内容、星级、国家与时间；保存后前台即时更新
   - 删除评论：在列表中点击“删除”，确认后移除该条评论
 - 建议：
   - 评论内容尽量图文并茂，提升可信度与转化率
   - 对低评分评论进行必要的售后跟进与说明，维护品牌形象

15、统计代码（Analytics）
 - 入口：Admin → Analytics
 - 内容：
   - HEAD 区域注入：粘贴第三方统计平台或验证脚本
   - Google 统计代码：单独粘贴 GA/GTM 片段
   - BODY 底部注入：部分平台建议插入的代码
 - 操作：粘贴代码后点击“保存设置”，全站生效

16、邮箱管理（Emails）
 - 入口：Admin → Messages 页面右上角“邮箱管理”（跳转到邮箱列表）
 - 搜索：在搜索框输入邮箱或姓名关键字，筛选当前列表
 - 复制：
   - 复制全部（用户名,邮箱）到剪贴板
   - 单条复制（用户名,邮箱）
 - 下载：
   - CSV（用户名,邮箱）
   - TXT（仅邮箱）
   - TXT（用户名,邮箱）
 - 刷新：点击“刷新”按钮更新列表数据

17、常见操作流程
 - 网站基础设置（首次上线）
   - 站点设置：在 `Settings` 填写站点名称、Logo、站点描述与关键词、联系方式、社媒链接、页脚与关于我们等文案，保存后前台生效
   - 首页内容：在 `Home Content` 编辑首页大标题、副标题、三张特色卡片文案
   - 导航菜单：在 `Navigation` 添加/排序导航项，启用后前台展示
   - 轮播图：在 `Carousel` 上传图片、设置按钮与跳转链接，调整排序并启用
   - SEO 设置：在 `SEO` 配置 Sitemap 与 Robots，填写站点验证；提示：标题/关键词/描述在 `Settings` 中配置
   - 统计代码：在 `Analytics` 的 `HEAD/Google/BODY` 区域粘贴第三方统计脚本，保存后全站生效
 - 新建产品（详细）
   - 准备分类：先在 `Categories` 创建或确认分类
   - 基本信息：填写产品名称（用于生成地址）、品牌与 UPC（可选）、上架时间（可选）
   - 价格与库存：设置售价与原价（可选），选择“有库存”或关闭库存标记
   - 图片：上传主图与图片集，支持批量上传与拖拽排序；建议首图清晰、尺寸统一
   - 要点（Bullet Points）：建议填写 5 条核心卖点，控制在合理长度内
   - 描述（Description）：完善图文描述，突出产品优势与使用场景
   - 亚马逊链接：填写以 `http/https` 开头的有效链接；如添加变体，主链接作为兜底跳转
   - 分类：选择所属分类（必填）
   - 展示选项：按需勾选“推荐产品（Featured）”“显示‘前往亚马逊’按钮”“显示‘加入购物车’按钮”
   - 变体（可选）：添加变体组与选项（如颜色、尺寸）；可为选项配置主图与购买链接；支持组合链接优先跳转
   - 保存与预览：点击保存后，前台“全部产品”可见；如勾选精选，首页精选区最多展示最新 3 条
 - 数据恢复：如需清空并恢复预设数据，在 `Reset` 选择 `Data` 或 `Full` 模式执行
 - 密码重置：忘记密码时在 `Reset` 使用 `Password` 模式重置为默认值

18、注意事项
 - 图片与链接尽量使用稳定的外链或平台存储，确保前台加载速度
 - 首页精选只显示 3 条，请优先设置需要露出的产品为精选
 - 如前台未显示数据，检查产品的 Active 与精选设置，以及是否存在分类关联

19、故障排除
 - 登录失败：
   - 使用默认账号 `dage666/dage168`
   - 尝试 Reset → Password 模式重置密码
 - 首页精选只有 1–2 条：
   - 确认至少有 3 个产品勾选了 Featured 且 Active 为启用
   - 新增或编辑产品后再次刷新首页
 - 前台不显示图片：
   - 检查图片链接有效性，或使用 Images 管理中上传的图片 URL
 - 修改后不生效：
   - 刷新页面；如仍不生效，检查是否保存成功或是否有必填项遗漏
